# Risk Assessment — RightToKnow Agent

## Risk Tier: Low

This agent assists citizens filing RTI applications under RTI Act 2005.
It provides informational assistance only and does not make legal decisions,
file on behalf of users, or store sensitive personal information.

### Key Controls
1. Clear disclaimer on every output — informational assistance, not legal advice
2. No PII storage — no Aadhaar, PAN, bank details stored in memory
3. Recommends licensed advocates for complex legal matters
4. All section citations drawn from actual RTI Act 2005 text
